# Torchup Agency

A landing page template dedicated to Social Media Marketing Agencies (SMMA) to intrduce their work and try to get in touch with customers.

## Tech Stack

* HTML5, CSS3
* Bootstrap v4
* JavaScript
* JQUERY
* Gulp.js
* fontAwesome
* Feather icons
